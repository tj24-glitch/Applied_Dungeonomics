Applied Dungeonomics Dataset

This dataset contains the cleaned and normalized character data used to calculate all indexes presented in the Applied Dungeonomics visualization.

All values are derived directly from raw character‑sheet attributes and standardized against absolute theoretical maxima for cross‑character comparability.

This dataset is intended for exploratory analysis, index replication, and independent inspection in spreadsheet or statistical software.

Download the Data

The full dataset is provided as a single CSV file containing raw attributes, normalized variables, alignment mappings, and computed index values.
